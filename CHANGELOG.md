# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-22

### Added
- Initial release of ML CLI
- Project initialization command (`ml init`)
- Exploratory data analysis command (`ml eda`)
- Data preprocessing command (`ml preprocess`)
- Model training with TPOT AutoML (`ml train`)
- Model prediction command (`ml predict`)
- FastAPI serving command (`ml serve`)
- Artifact cleanup command (`ml clean`)
- Shell completion support (`ml completion`)
- Comprehensive documentation
- GitHub Actions CI/CD pipeline
- Multi-OS and multi-Python version testing
- Integration tests
- API endpoint tests
